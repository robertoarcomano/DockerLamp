# DockerLamp
How to build a Lamp working using Docker

## Create a directory
```mkdir DockerLamp```

## Enter the directory
```cd DockerLamp```

## Download and execute download.sh script
```wget https://raw.githubusercontent.com/robertoarcomano/DockerLamp/master/install.sh -q -O -|bash```
